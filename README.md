# previsaoBTCapi
 
Bitcoinpredict AI Model é um modelo de Machine Learning treinado com toda a série histórica do Bitcoin, utilizando dados de preço, volume, indicadores técnicos e informações temporais para predizer o valor do BTC na próxima hora.

O modelo é alimentado por features como Open, High, Low, Volume, além de indicadores técnicos amplamente utilizados no mercado, incluindo RSI, EMA, Bandas de Bollinger, MACD, médias de volume e variáveis temporais (ano, mês, dia, dia da semana, hora e minuto).

Com base nos indicadores atuais do mercado, o modelo realiza previsões de curto prazo, sendo ideal para análise quantitativa, estudos de mercado, trading algorítmico e pesquisa financeira.
